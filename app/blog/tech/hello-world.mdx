# Next.js와 MDX로 기술 블로그 만들기

안녕하세요! 이 포스트는 Next.js와 MDX를 사용하여 기술 블로그를 만드는 방법에 대해 설명합니다.

## MDX란?

MDX는 마크다운에 JSX를 추가한 포맷입니다. 다음과 같은 특징이 있습니다:

- 마크다운 문법 사용 가능
- React 컴포넌트 임포트 및 사용
- JavaScript 표현식 사용

## 코드 예제

```typescript
const sum = (a: number, b: number): number => {
  return a + b;
};

const multiply = (a: number, b: number): number => {
  return a * b;
};
```
